 include
         Ye method ek string ya array mein kisi specific substring ko search karne ke liye use hota hai.

         Function: Ye ek general block of code hota hai jo koi task perform karta hai aur aap usse directly call kar sakte ho (jaise function myFunction() { ... }).

        Method: Ye ek function hota hai jo kisi object ke saath associated hota hai (jaise element.addEventListener, jo ek method hai jo DOM elements ke saath use hota hai).

        Matlab, function ko aap independent tarike se use kar sakte ho, jabki method ek object ke andar hota hai aur us object ke saath hi kaam karta hai.



addEventListener
     addEventListener ek method hai jo aapko kisi element (jaise button, input field, etc.) par hone wale events (jaise click, key press, etc.) ko handle karne ke liye ek function add karne ka tareeka deta hai.
     jab wo event hota hai, to aapka diya hua function run hota hai.
